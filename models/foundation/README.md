# Foundation Models

Upstream foundation-model weights are not tracked in this GitHub repository.
Retrieve implementation and release information from the upstream GitHub
repositories, and place local weight files here only for local reuse:

| Expected local file | Upstream source |
| --- | --- |
| `mace-omat-0-medium.model` | MACE code: <https://github.com/ACEsuit/mace>; MACE foundation models: <https://github.com/ACEsuit/mace-foundations> |
| `DPA-3.1-3M.pt` | DeePMD-kit / DPA model implementation: <https://github.com/deepmodeling/deepmd-kit> |

The fine-tuned model files generated for this study are stored in
`../fine-tuned/`.
