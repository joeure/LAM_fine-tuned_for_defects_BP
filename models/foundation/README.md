# Foundation Models

Upstream foundation-model weights are not tracked in this GitHub repository.
Retrieve them from their original sources and place them here only for local
reuse:

| Expected local file | Upstream source |
| --- | --- |
| `mace-omat-0-medium.model` | MACE foundation models: <https://github.com/ACEsuit/mace-foundations> |
| `DPA-3.1-3M.pt` | DPA-3.1-3M model page: <https://www.aissquare.com/models/detail?id=343&name=DPA-3.1-3M&pageType=models>; DeePMD-kit code: <https://github.com/deepmodeling/deepmd-kit> |

The fine-tuned model files generated for this study are stored in
`../fine-tuned/`.
